What is GCC?

GCC stands for GNU Compiler Collection. It can be used for compiling code in various languages, though the two most common are C and C++. Typically, the gcc command is used for compiling code in C, and the g++ command is used for compiling code in C++. However, since C is backwards-compatible with C++, you can use the g++ command to compile C code, though the computer will read it as a C++ file.

How to use gcc

First, open your command line and navigate to the directory where you have your C or C++ file. The type in the following command: gcc filename.c
After typing this command, there will be a new file in your directors named a.out. Type .\a on your command line to execute the file.
